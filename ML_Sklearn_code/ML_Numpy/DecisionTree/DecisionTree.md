## 简介

决策树模型就是数据结构中的树，根据**特征选择依据**(信息熵)等划分特征，生成决策树，然后**剪枝**提高泛化能力，可分类可回归，代表算法有ID3，C4.5和CART

## 优缺点

优点：计算复杂度不高，输出结果易于理解，对中间值的缺失不敏感，可以处理不相关特征数据。

缺点：可能会产生过度匹配问题

适用数据类型：数值型和标称型

## 决策树生成算法

|      | 特征选择依据 |      |
| :--: | :----------: | ---- |
| ID3  |    信息熵    |      |
| C4.5 |  信息增益比  |      |
| CART |   基尼系数   |      |
